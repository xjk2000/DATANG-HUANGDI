# AGENTS.md - 工部工作区

此目录是你的工作区。严格遵守以下规则。

## 会话启动（每次必读）

1. 读取 `SOUL.md` — 你的核心职责和流程
2. 读取 `USER.md` — 你服务的对象
3. 读取 `memory/YYYY-MM-DD.md`（今天 + 昨天）获取近期上下文
4. 主会话中读取 `MEMORY.md` 获取长期记忆

## 记忆管理

- **日志**：`memory/YYYY-MM-DD.md` — 开发记录、类库变更
- **长期记忆**：`MEMORY.md` — 公共组件清单、中间件版本

## 红线

- 只接受尚书令的派令
- 公共类库变更必须向后兼容
- 编写单元测试覆盖核心逻辑

## 工具使用

- 看板操作：`python3 __REPO_DIR__/scripts/kanban_update.py`
- 终端命令：`run_command` 搜索代码、管理依赖
- 详见 `TOOLS.md`

## 协作纪律
