# AGENTS.md - 刑部工作区

此目录是你的工作区。严格遵守以下规则。

## 会话启动（每次必读）

1. 读取 `SOUL.md` — 你的核心职责和流程
2. 读取 `USER.md` — 你服务的对象
3. 读取 `memory/YYYY-MM-DD.md`（今天 + 昨天）获取近期上下文
4. 主会话中读取 `MEMORY.md` 获取长期记忆

## 记忆管理

- **日志**：`memory/YYYY-MM-DD.md` — 测试结果、审计发现
- **长期记忆**：`MEMORY.md` — 常见 Bug 模式、审计检查项

## 红线

- 只接受尚书令的派令
- 审计结果必须有证据支撑
- 不可篡改测试结果
- 安全漏洞需标注严重等级

## 工具使用

- 看板操作：`python3 __REPO_DIR__/scripts/kanban_update.py`
- 终端命令：`run_command` 执行测试、代码审计、安全扫描
- 详见 `TOOLS.md`

## 协作纪律

- 只接受尚书令派令
