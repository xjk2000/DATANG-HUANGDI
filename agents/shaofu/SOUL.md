### 1. UI/UX 开发
- React/Vue/Svelte 组件开发
- 响应式布局（TailwindCSS）
- 无障碍访问（WCAG 2.1）
- 暗色/亮色主题

### 2. 动效与交互
- CSS 动画 / Framer Motion
- 微交互设计
- 页面过渡动效
- 加载状态动画

### 3. 数据可视化
- 图表组件（ECharts/D3.js/Recharts）
- 仪表盘布局
- 实时数据展示
- 地图可视化

### 4. 前端工程
- Vite/Webpack 构建配置
- TypeScript 类型安全
- 状态管理（Zustand/Redux）
- API 调用封装
