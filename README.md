# ⚔️ 帝王系统 · Claw DiWang

> 中书取旨，门下封驳，尚书奉而行之。
> —— 以唐制三省六部五监，驾驭 AI 多 Agent 协作。

基于 [OpenClaw](https://openclaw.ai) 构建的多 Agent 协作编排系统。17 个专职 AI Agent 组成完整的三省六部五监体系，以制度性审核、分权制衡、实时可观测为核心，实现复杂任务的可靠执行。

## 🎯 17 个 Agent 全景

### 三省（决策层）
| Agent | ID | 职责 |
|-------|-----|------|
| **中书令** | `zhongshuling` | 接收皇帝旨意，拆解任务，起草敕令 |
| **中书舍人** | `zhongshu_sheren` | 记录旨意，辅助中书令分析任务 |
| **侍中侍郎** | `shizhong` | 审查敕令，决策准奏或封驳 |
| **给事中** | `jishizhong` | 排查敕令漏洞，驳正违失 |
| **尚书令** | `shangshuling` | 接收准奏敕令，分析并派发到六部/五监 |

## 📋 敕令状态流转

```
皇上 → 中书取旨 → 中书起草 → 门下审议 → 准奏派发 → 执行中 → 待审查 → ✅ 完成
                                ↑          │

```

状态机严格校验，非法跳转被拒绝并记录日志。

## 🚀 快速开始

### 前置条件
- [OpenClaw](https://openclaw.ai) 已安装
- Python 3.9+
- macOS / Linux

### 安装
```bash
git clone <repo-url> claw-diwang
cd claw-diwang
chmod +x install.sh && ./install.sh
```

### 启动
```bash
# 终端 1：数据刷新循环
bash scripts/run_loop.sh

# 终端 2：看板服务器
python3 dashboard/server.py

# 打开浏览器
open http://127.0.0.1:7891
```
